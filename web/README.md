# ReliefConnect - Disaster Relief Web Application

A comprehensive, user-centric disaster relief platform designed to provide immediate assistance, community support, and emergency preparedness resources during crisis situations.

## üåü Features

### Core Functionality

- **AI-Powered Relief Recommendations**: Intelligent system that provides personalized resource suggestions based on user needs
- **Quick Needs Selection**: Streamlined interface with common emergency needs (Food & Water, Shelter, First Aid, Power/Charging, etc.)
- **Geolocation Integration**: Automatic address detection for faster order processing
- **Real-time Inventory Status**: Live updates on resource availability and supply levels

### Purpose-Driven Pages

- **Information Hub**: Emergency contacts, first aid guides, live disaster updates
- **Community Board**: Peer-to-peer help network with "I Need Help" and "I Can Help" sections
- **Volunteer Portal**: Volunteer opportunities and donation links to trusted organizations
- **Disaster Preparedness**: Interactive checklists for emergency kits with progress tracking

### Global Enhancements

- **Multi-Language Support**: Language switcher for English, Spanish, and French
- **Mark Safe Feature**: Quick safety status updates for family and friends
- **Accessibility Features**: High contrast mode, reduced motion support, screen reader compatibility
- **Mobile-First Design**: Responsive layout optimized for all device sizes

### Phase 2 Placeholders

- **Order Tracking**: Real-time delivery monitoring (Coming Soon)
- **Fraud Reporting**: Comprehensive fraud detection and reporting system

## üöÄ Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn package manager

### Installation

```bash
# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

### Available Scripts

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run preview  # Preview production build
npm run lint     # Run ESLint
npm run test     # Run tests
```
